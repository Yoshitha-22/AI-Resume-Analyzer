# AI Resume Analyzer

AI Resume Analyzer is a web application built using Python and Streamlit that allows users to upload a PDF resume and extract its contents automatically.

## Features

* Upload resume in PDF format
* Extract text from resumes using PyPDF2
* Interactive web interface using Streamlit
* Foundation for ATS score analysis and AI-powered feedback

## Technologies Used

* Python
* Streamlit
* PyPDF2

## How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run the application:

streamlit run app.py
